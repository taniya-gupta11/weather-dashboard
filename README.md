## Project Overview
The **Weather Dashboard** is a web application that allows users to:
- View the **current weather** of any city.
- Check a **5-day weather forecast**.
- Save the **last searched city** using Local Storage.
- Handle **loading states** and **errors** gracefully.

This project demonstrates **JavaScript, async/await, REST API integration, Local Storage, and DOM manipulation** skills.
